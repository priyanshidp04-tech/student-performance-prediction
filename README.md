# 🎓 Student Performance Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict student performance (final marks) using Machine Learning techniques based on key academic and lifestyle factors.

---

## 🎯 Objective

To build a predictive model that estimates student marks using:

* Study Hours
* Attendance
* Previous Marks
* Sleep Hours

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🧠 Algorithm Used

* Linear Regression

---

## 📊 Dataset

A synthetic dataset containing student-related features:

* study_hours
* attendance
* previous_marks
* sleep_hours
* final_marks

---

## 🚀 Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training (Linear Regression)
6. Prediction
7. Model Evaluation (R² Score, MSE)
8. Visualization

---

## 📈 Model Performance

* **R² Score:** ~0.89
* **Mean Squared Error:** Low

➡️ The model performs well and provides reliable predictions.

---

## 🔍 Key Insights

* Previous marks and study hours are the most influential factors.
* Higher attendance improves performance.
* The model predictions closely match actual values.
* Student performance depends on consistency and effort.

---

## ⚠️ Limitations

* Dataset is small and synthetic
* Real-world factors like stress and environment are not included

---

## 🚀 Future Improvements

* Use real-world datasets
* Apply advanced models (Decision Tree, Random Forest)
* Add more features

---

## 💡 Conclusion

The model successfully predicts student performance with high accuracy and demonstrates the effectiveness of Machine Learning in educational analytics.
